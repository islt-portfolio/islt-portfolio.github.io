Accessibility Statement

Commitment to Accessibility

This portfolio is committed to ensuring digital accessibility for people with disabilities. I am continually working to improve the user experience for all visitors and apply relevant accessibility standards.

Conformance Status

This website strives to conform to Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible to people with disabilities.

Accessibility Features

This site includes the following accessibility features:

  • Semantic HTML markup for proper document structure
  • Skip navigation link for keyboard users
  • Descriptive alt text for images
  • Adequate color contrast ratios
  • Responsive design that works across devices
  • Keyboard-navigable interactive elements
  • Closed caption and fall-back option for video.
  • Clear and consistent navigation

Compatibility

This website has been tested for compatibility with:

  • Current versions of Chrome, Firefox, Safari, and Edge browsers
  • Windows and Mac operating systems
  • Mobile devices (Android-simulator and Apple iOS)
  • Screen readers: VoiceOver compatibility

Feedback

I welcome feedback on the accessibility of this portfolio. If you encounter any accessibility barriers or have suggestions for improvement, please contact me:

  • Email: islt_portfolio@icloud.com

I will make every reasonable effort to respond to accessibility concerns within 5 business days.

Technical Specifications

This website relies on the following technologies for accessibility:

  • HTML5
  • CSS3
  • Web browsers with CSS and HTML5 support
